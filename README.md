Запустить собранный контейнер и проверить, что он запустился от вашего пользователя. Для этого необходимо выполнить Docker exec -ti <container_id> bash в соседеней консоли, предварительно узнав идентификатор запущенного контейнера. Назовите все идентификаторы процессов для родительского и порожденного процессов. 

Родительский (nginx master) - 1 root
